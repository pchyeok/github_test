- mybatis-config.xml : DB 테이블, 컬럼명 properties 설정, Underscore -> Camel 명명법으로 매핑
- board Mapper : namespace 경로 설정 및 쿼리 작성
- reply Mapper : namespace 경로 설정 및 쿼리 작성 

- domain : boardVO, replyVO 클래스, using field, getter/setter, toString 설정

(CRUD 작업, @Service - DB상관없이 데이터 처리 및 직접 통신 하지않게 설정)

@RequestMapping 설정 (각 경로 board, reply, replies, member)
